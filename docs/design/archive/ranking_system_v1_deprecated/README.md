## 🔄 方案演进历史

### v1.0 - 早期方案（已归档）

**算法**: 截尾均值(10%) + 贝叶斯平滑(k=20)

**优点**:
- ✅ 冷启动保护好
- ✅ 学术界认可

**缺点**:
- ❌ 缺少用户权重机制
- ❌ 可解释性差（贝叶斯平滑难以解释）
- ❌ 参数调优困难（第一届无历史数据）

**归档时间**: 2025-01-07

---

### v2.0 - 混合方案（当前生效）✅

**算法**: 用户权重(W=1+ln(1+n)) + 双侧截尾(10%) + 算术平均

**优点**:
- ✅ 公平性提升（用户权重）
- ✅ 可解释性强（简单算术平均）
- ✅ 第一届可行（无历史数据依赖）
- ✅ 符合12榜单架构

**缺点**:
- ⚠️ 刷分>20%时截尾失效（需人工监控）
- ⚠️ 无冷启动保护（第二届引入贝叶斯平滑）

**生效时间**: 2025-01-07
