## 🚀 使用方式

### 日常使用流程

```yaml
每天开始:
  "开始今天的工作"
  → workflow-orchestrator-agent
  → 智能推荐今日任务
  → 自动启动相应Agent

设计讨论:
  "继续讨论"
  → discussion-agent
  → 自动加载问题
  → 自动记录决策
  → 询问是否继续

模块完成:
  "完成这个模块"
  → workflow-orchestrator-agent
  → 自动协调验证步骤
  → 确保质量标准

代码审核:
  "审核这段代码"
  → code-review-agent
  → 5维代码审核
  → 具体修复建议

每天结束:
  "生成工作日报"
  → workflow-orchestrator-agent
  → 总结今日完成
  → 规划明日任务
```

### Agent协作关系

```yaml
中心协调者: workflow-orchestrator-agent
  ↓ 调用和协调
  ↓
设计阶段:
  ├─ discussion-agent (讨论问题)
  ├─ design-audit-agent (审核设计)
  └─ completion-check-agent (验证完整性)

开发阶段:
  ├─ code-review-agent (审核代码)
  └─ completion-check-agent (验证完整性)

文档管理:
  └─ doc-sync-agent (同步文档)

反馈:
  所有Agent → workflow-orchestrator-agent
  → 更新状态 → 推荐下一步
```
