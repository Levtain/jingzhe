## ⚡ 快速参考

### 核心算法(一句话版本)

**截尾均值(10%) + 贝叶斯平滑(k=20)**

> 去掉最高和最低各10%的极端评分,然后对小样本向全局均值回归(k=20)

### 关键参数速查

| 参数 | 值 | 说明 |
|------|-----|------|
| **min_ratings** | 20人次 | 排名资格门槛 |
| **trim_percent** | 10% | 截尾比例 |
| **bayesian_k** | 20 | 平滑强度参数 |
| **tie_epsilon** | 0.01分 | 并列判定阈值 |

### 4个关键修正

1. ✅ 贝叶斯平滑使用"人次"(评审团×2)而非"人数"
2. ✅ 全局均值包含所有有评分的游戏
3. ✅ 截尾百分比保持10%
4. ✅ 并列epsilon调整为0.01
