# 惊蛰计划 (Jingzhe Project)

## 项目简介

惊蛰计划是一个游戏竞赛平台的完整设计文档项目，旨在为游戏开发者提供一个公平、有趣、互动性强的竞赛环境。

## 项目结构

```
d:/Claude/
├── docs/                   # 文档目录
│   ├── design/            # 设计文档
│   ├── product/           # 产品文档
│   └── questions.md       # 问题清单
└── README.md              # 本文件
```

## 核心文档

### 设计文档
- **关键名词解释索引**: 所有系统概念的中央词汇表
- **评分系统设计**: 完整的评分、榜单、成就系统设计
- **用户角色与权限**: 5种角色的权限定义和页面级控制
- **团队系统设计**: 团队组建、角色分配、时间线管理
- **页面功能设计**: 8个核心页面的完整功能设计
- **社区界面设计**: 社区互动、内容管理、审核系统
- **经济系统设计**: 打赏、推荐位、成就奖励等经济模型

### 产品文档
- **claude.md**: Claude AI配置文件
- **CHANGELOG.md**: 版本更新记录
