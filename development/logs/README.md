# 开发日志

存放惊蛰计划相关的工作日志和沟通记录。

---

## 📂 目录结构

### communications/ - 惊蛰计划沟通记录

- **with-candle/** - 和产品经理(蜡烛先生)的沟通
- **with-team/** - 和团队的沟通
- **meetings/** - 会议记录

### workflow/ - 工作流日志

- 工作流优化记录
- 自动化工具使用日志

### session-end/ - 会话结束日志

- 每日会话总结
- 会话结束自动生成的报告

### context-monitor/ - 上下文监控日志

- 上下文使用率监控
- 自动快照触发记录

---

## ⚠️ 重要提示

**非惊蛰计划的沟通记录** 请存放在项目根目录的 `communications/` 文件夹:

- **communications/general/** - 通用沟通
- **communications/learning/** - 学习相关
- **communications/other-projects/** - 其他项目

## 📝 分类原则

### 应该放在这里 (development/logs/communications/)

- ✅ 进度汇报
- ✅ 产品经理沟通
