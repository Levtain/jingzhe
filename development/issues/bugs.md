## 📝 Bug 报告模板

当发现 Bug 时，请按以下格式报告：

```
### [Bug标题]

**发现时间**：YYYY-MM-DD
**发现人**：蜡烛先生 / Claude

**问题描述**：
简单描述遇到了什么问题

**复现步骤**：
1. 第一步做什么
2. 第二步做什么
3. 出现了什么结果

**期望结果**：
应该是什么样

**实际结果**：
实际是什么样

**截图/录屏**：
（如果有）

**优先级**：
- 🔴 严重（阻塞使用）
- 🟡 一般（影响使用）
- 🟢 轻微（体验问题）
```

## 🔍 如何报告 Bug

### 如果你是蜡烛先生（产品经理）

**直接告诉我**：
```
【Bug报告】
我在测试XX功能时，遇到了问题：
（描述你看到的、听到的、感觉不对的地方）

我当时是在这样操作的：
1. ...
2. ...

我期望的是：...
但实际发生的是：...
```

**不用担心**：
- ✅ 不需要技术术语
- ✅ 不需要知道是什么原因
- ✅ 只要说清楚"看到什么"和"期望什么"
