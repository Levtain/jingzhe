- **competitors/** - 竞品分析

- **industry/** - 行业报告

  - 用户研究报告

### references/ - 参考资料

- **articles/** - 参考文章
- **books/** - 参考书籍
- **standards/** - 标准/规范

### tools/ - 工具资料

### assets/ - 媒体资产
