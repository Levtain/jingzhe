# 惊蛰计划 (Jingzhe Project)

## 项目简介

惊蛰计划是一个游戏竞赛平台的完整设计文档项目，旨在为游戏开发者提供一个公平、有趣、互动性强的竞赛环境。

---

## 📁 重要：协作者必读

### 🎯 只需要关注这两个文件夹！

**协作者只需要关注以下两个文件夹**：

```
d:/Claude/
├── docs/                   # ✅ 产品和设计文档（重要！）
│   ├── design/            # 系统设计文档
│   │   ├── 评分系统设计方案_v*.md
│   │   ├── 用户角色与权限_*.md
│   │   ├── 团队系统设计_*.md
│   │   ├── 页面功能设计_*.md
│   │   ├── 社区界面设计_*.md
│   │   └── 经济系统设计_*.md
│   └── product/           # 产品文档
│       ├── claude.md      # AI配置和协作规则
│       └── CHANGELOG.md   # 版本更新记录
│
└── development/           # ✅ 工作文档（重要！）
    ├── active/            # 当前活跃项目
    │   ├── issues/       # 待确认问题清单
    │   ├── planning/     # 项目规划
    │   └── tracking/     # 进度跟踪
    ├── logs/             # 工作日志
    └── memories/         # 项目记忆库
```

### 📂 其他文件夹说明

**不需要关注**（内部工具配置）：

- `.claude/` - Claude AI 工具配置（内部使用）
- `.config/` - 本地配置文件
- `input/` - 临时输入区
- `resources/` - 参考资料
- `development/workspace/` - 临时工作区
- `development/archive/` - 归档文件

**总结**：
- ✅ **看 `docs/` 和 `development/active/`** - 这里有所有你需要的内容
- ❌ **忽略其他文件夹** - 那些是工具内部配置，不需要关注
