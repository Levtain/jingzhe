## 🔥 核心功能1：话题讨论区 (新增)

### 设计理念
参考机核网的"话题讨论"功能,打造UGC驱动的话题社区。

### 1.1 话题列表页

#### 导航筛选
**顶部标签栏**：
- 热门话题
- 最新话题
- 我关注的话题
- 我参与的话题
- 我创建的话题

**筛选工具栏**：
- **话题类型**：[全部] [游戏开发] [游戏设计] [美术分享] [音乐音频] [引擎技术] [行业讨论] [活动相关]
- **排序方式**：[最热] [最新] [精华] [最多评论]
- **搜索框**：按话题标题/内容/作者搜索

#### 话题卡片设计
**基础信息**：
- 话题标签（显示类型，如"游戏开发"）
- 话题标题
- 作者头像 + 昵称 + 身份标识
- 发布时间
- 封面图（可选，如分享游戏开发图片）

**互动数据**：
- 👁️ 浏览数
- 💬 评论数
- 👍 点赞数
- ⭐ 收藏数

**状态标识**：
- 🔥 热门话题（浏览数>1000或评论数>50）
- ⭐ 精华话题（管理员或社区推荐）
- 💬 我参与过（已评论）
- 👁️ 我看过（已浏览）

**内容预览**（2行）：
- 显示话题前两行内容
- 或话题摘要（作者可编辑）

### 1.2 话题详情页

#### 内容区
**话题头部**：
- 话题标签
- 话题标题
- 作者信息栏：
  - 头像 + 昵称 + 身份标识
  - 注册时间 + 作品数
  - "关注作者"按钮
- 发布时间 + 最后更新时间
- 编辑按钮（作者本人可见）

**正文内容**：
- 支持Markdown渲染
- 支持图片、视频、音频嵌入
- 支持代码高亮（技术讨论）
- 附件下载区（如分享工具包）

**话题元信息**：
- 所属话题分类
- 相关话题推荐（自动计算相似话题）
- 话题统计：浏览、评论、点赞、收藏数据

**操作区**：
- 点赞 👍
- 收藏 ⭐
- 分享 📤（复制链接、社交媒体分享）
- 举报 🚩
- "参与讨论"按钮（滚动到评论区）

#### 评论区（增强版⭐）
**评论列表**：
- 每条评论显示：
  - 评论者头像 + 昵称 + 身份标识
  - 评论内容（支持简单Markdown）
  - 评论时间
  - 点赞数
  - 回复按钮
  - 打赏按钮（0-5金币）
  - "引用评论"按钮（引用到评论框）

**评论排序**：
- [最新] [热门] [作者回复优先]

**评论输入区**（增强）：
- **工具栏**：
  - 粗体、斜体、引用、代码、链接
  - 上传图片按钮
  - 引用话题内容按钮
  - 表情选择器
- **字数提示**：已输入 X/2000字
- **发布按钮**："发表评论"

**二级回复**：
- 点击"回复"展开二级回复列表
- 二级回复缩进显示
- 支持多级回复（建议≤3级）

**高亮功能**⭐：
- 作者回复的评论：高亮显示"作者回复"标识
- 高赞评论：点赞数>10，高亮显示
- 开发者评论：如有游戏作者参与，显示"开发者"标识

### 1.3 创建/编辑话题

#### 创建话题入口
- 社区首页："创建话题"按钮（右上角）
- 话题列表页：悬浮"＋"按钮（右下角，类似机核网APP）

#### 创建话题表单
**必填项**：
- **话题标题**（5-100字）
- **话题类型**（下拉选择）
  - 游戏开发、游戏设计、美术分享、音乐音频、引擎技术、行业讨论、活动相关
- **话题内容**（支持Markdown编辑器）

**可选项**：
- **封面图**（上传，16:9比例）
- **附件**（上传文件，如工具包、素材包）
- **相关游戏**（关联到已提交的游戏，可选）
- **话题标签**（自定义标签，最多3个）

**高级选项**（折叠）：
- **允许评论**：[是] [否]（默认是）
- **话题可见性**：[公开] [仅参赛者可见] [仅评审团可见]
- **置顶时长**：管理员可设置置顶

**编辑器功能**⭐：
- 实时预览（左右分栏）
- 工具栏：加粗、斜体、标题、引用、代码、链接、图片、视频、附件
- 字数统计
- 自动保存草稿（每30秒）

### 1.4 话题推荐机制（智能算法）

**推荐算法逻辑**：
1. **协同过滤**：推荐我关注的话题类型下的热门话题
2. **内容相似度**：基于话题标题和内容计算相似度
3. **社交关系**：推荐我关注的作者发布的话题
4. **时间衰减**：新话题优先，但高质量老话题也可长期展示

**推荐位置**：
- 话题列表页顶部："为你推荐"专区（3个话题）
- 话题详情页侧边栏："相关话题"推荐（5个话题）

- **团队风格**（单选）：[严肃认真] [轻松愉快] [学习导向] [冲刺作品]

  - 目标完成时间（日期选择）
- **沟通方式**：
  - 平台选择（下拉：Discord/QQ/微信/站内私信）

- **技能需求明细**（动态添加）：

**预览功能**：
- "预览发布效果"按钮（实时预览卡片展示效果）

### 2.3 申请与沟通（参考机核网"机组"协作）

**申请状态**：
- 申请者可在"我的申请"中查看状态：
  - ⏳ 待处理
  - ✅ 已接受
  - ❌ 已拒绝
- 队长可在"组队管理"中管理所有申请

#### 团队协作区（新增⭐）
**仅已组建团队可见**：
- **团队内部聊天室**：
  - 文字聊天
  - 文件共享
  - 语音通话入口（接入第三方服务如Discord）
- **任务看板**（可选）：
  - 待办、进行中、已完成三列
  - 每个任务可分配给成员
  - 简易版Trello
- **日程安排**：
  - 开发时间规划表

- **代码仓库集成**（可选）：

## 🎨 UI/UX 优化建议

### 1. 设计原则（继承机核网风格）
- **内容优先**：突出内容本身，弱化装饰
- **渐进式展示**：信息按需展开，避免信息过载
- **视觉层次**：清晰的信息层级，重要信息突出
- **品牌一致性**：与惊蛰计划整体风格统一

### 2. 交互优化
- **无限滚动**：话题列表、评论列表采用无限滚动
- **快捷操作**：
