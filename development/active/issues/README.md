## 🔄 工作流程

### 设计讨论阶段

```
1. 查看当前待讨论问题
   → 打开 questions.md

2. 开始问题讨论
   → 使用 /discuss 命令

3. 记录讨论结果
   → 标记已确认问题(✅)

4. 同步到设计文档
   → 使用 /sync-docs 命令

5. 检查完成情况
   → 使用 /check-completion 命令
```

### 开发阶段

```
1. 发现Bug
   → 记录到 bugs.md

2. 修复Bug
   → 更新状态为"已修复"

3. 验证修复
   → 标记为"已验证"

4. 归档Bug记录
   → 移动到 archive/
```
